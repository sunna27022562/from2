Web 3.0 คืออะไร?

Web 3.0 คือ แนวคิดการใช้งานเว็บไซต์ที่เอื้อให้ผู้ใช้งานถือสิทธิความเป็นเจ้าของสินทรัพย์ดิจิทัลได้อย่างสมบูรณ์ เดิมที
เรียกว่าSemantic Web ซึ่งเชื่อมโยงข้อมูลต่าง ๆ ด้วยเทคโนโลยีที่ช่วยให้ผู้ใช้งานสร้างสิ่งต่าง ๆ ด้วยตัวเอง ทั้พื้นที่จัดเก็บข้อมูล โครงสร้างฐานข้อมูล และกฎที่ใช้จัดการข้อมูล
ทั้งนี้ แนวคิด Web 3.0 แบ่งออกเป็น 2 มุมมอง ได้แก่ มุมมองในอดีต และมุมมองปัจจุบัน โดยแนวคิด Web 3.0 ในมุมมองอดีตอ้างอิง
จากTim Berners-Lee ผู้คิดค้น World Wide Web มุ่งเน้นไปที่องค์ประกอบ 3 อย่าง ได้แก่ Autonomous, Intelligent และ Open Internet
ส่วนแนวคิด Web 3.0 ในมุมมองปัจจุบันอ้างอิงจากข้อมูลของ Dr. Gavid Wood อดีต CTO ของ Ethereum และผู้ก่อตั้ง Web 3.0
Foundation ระบุว่า Web 3.0 ว่าด้วยอินเทอร์เน็ตไร้เซิร์ฟเวอร์และกระจายศูนย์ ผู้ใช้งานทุกคนควบคุมข้อมูล ตลอดจนตัวตนในโลกดิจิทัลได้ด้วยตัวเอง
