Web 3.0 ที่จะเกิดขึ้นในอนาคต

Sergey Nazarov หนึ่งในผู้ร่วมก่อตั้ง Chainlink ได้กล่าวถึงทิศทางของ Web 3.0 ในหัวข้อการนำเสนอ “Future of Chainlink” ว่า ในช่วงที่ผ่านมาเทรนด์ที่ได้รับความสนใจอย่างมากในโลกของ Web 3.0 คือ DeFi (Decentralized Finance) และ NFT (Non-fungible Token) ซึ่งเป็นช่วงเวลาที่ได้นำการเปลี่ยนแปลงมาสู่ผู้ใช้งานอินเทอร์เน็ต และในอนาคต เมื่อผู้ใช้งานและสถาบันต่าง ๆ เริ่มให้การยอมรับมากขึ้น ย่อมส่งผลต่อคุณภาพของการเป็นเครือข่ายที่ Open, Transparent, Reliable และ Decentralized ให้มีความแข็งแกร่งขึ้นไปด้วย

Web 3.0 ถือเป็นการเดินทางของรูปแบบอินเทอร์เน็ตที่น่าสนใจไม่น้อย เพราะการเปลี่ยนผ่านจากการใช้งานเว็บไซต์แบบ Interactive ใน Web 2.0 สู่การเป็นเจ้าของสินทรัพย์ดิจิทัลอย่างสมบูรณ์ใน Web 3.0 นั้น  ทำให้ผู้ใช้งานอินเทอร์เน็ตควบคุมข้อมูลและสินทรัพย์ของตนได้ โดยปราศจากตัวกลางเข้ามาแทรกแซงประเด็นดังกล่าว
