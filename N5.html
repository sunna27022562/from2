ข้อจำกัดของ Web 3.0

 
 
 
 
 
Web 3.0 ยังมีข้อจำกัดด้านความสามารถในการขยาย (Scalability) อยู่ เป็นเหตุให้การดำเนินการธุรกรรมช้ากว่าเว็บไซต์แบบ Centralized เนื่องจากความเป็น Decentralized
ย่อมหมายถึงทุกครั้งที่มีการทำธุรกรรมเพื่อชำระค่าสินค้าและบริการนั้นจะต้องรอการดำเนินการของ Validator หรือ โหนด (Node) จากทั่วทั้งเครือข่ายด้วยผู้ใช้งานเข้าถึงการใช้งาน
Web 3.0 ได้ยาก เนื่องจากกลุ่มผู้ใช้งาน Web Browser ในปัจจุบัน ยังคงยึดติดและเชื่อถือการใช้บริการจากแพลตฟอร์ม Web 2.0 มากกว่าเทคโนโยลีที่สามารถทำงานร่วมกับ Web 3.0 ยังมีไม่มากนัก
