Blockchain สำคัญอย่างไรต่อ Web 3.0?

เป็นที่ทราบกับว่า Blockchain ใช้ Cryptography และระบบ Consensus ที่แตกต่างกันออกไปเพื่อดำเนินการตรวจสอบข้อมูลและรักษาความปลอดภัยของเครือข่าย
แต่ไม่ว่าจะใช้กลไกการทำงานแบบใด การทำงานของ Blockchain จะยึดถือหลักของการเปิดกว้าง ไม่มีตัวกลาง การใช้เทคโนโลยีที่มีความโปร่งใส ความปลอดภัยและความเป็นส่วนตัว ซึ่งการพัฒนา Web 3.0 จึงต้องพึ่งพาเทคโนโลยี Blockchain ในการแก้ปัญหาข้อด้อยของอินเทอร์เน็ตในปัจจุบันที่ถูกผูกขาดโดยบริษัทเทคโนโลยีต่าง ๆ โดยที่ยังสามารถทำงานควบคู่ไปกับ Semantic Web ได้ ยกตัวอย่างเช่น การ Governance เป็นวิธีการที่แพร่หลายบนแพลตฟอร์มประเภท DApp (Decentralized Application) และเป็นส่วนที่ได้รับการยอมรับจากชุมชนผู้ใช้งานเครือข่ายเพราะการมีส่วนร่วมการในกำหนดทิศทางของเครือข่ายหรือโปรโตคอลนั้นเป็นสิ่งแสดงถึง Decentralization ของแพลตฟอร์มซึ่งเป็นตัวอย่างของการนำจุดเด่นของเทคโนโลยี Blockchain มาใช้บน Web 3.0 อย่างไรก็ตาม น้ำหนักของการโหวตสำหรับการ Governance นั้นขึ้นอยู่กับโทเคนที่ถือครองอยู่ จึงต้องพิจารณาในส่วนนี้ด้วยเช่นกัน
